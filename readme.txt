Advanced meta widget allows you to chose, what you want to display in meta widget and adds many other features to manage it.

== Description ==

Advanced meta widget adds new widget which does the same as basic Wordpress built-in meta widget but allows you to chose what you want to display, and adds more features for managing this widget.

== Installation ==

Unpack archive contents into plugin folder (/wp-content/plugins/) and activate it in Wordpress admin. Go to Appearance -> Widgets and add Advanced meta widget to your widget area. Then set all settings to what you want. Enjoy :)
